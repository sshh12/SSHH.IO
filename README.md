# SSHH.IO

Personal website for [Shrivu Shankar](https://sshh.io). Built with Vite + React, hosted on Netlify.

## Development

```bash
npm install
npm run dev       # Start dev server at localhost:5173
npm run build     # Production build → dist/
npm run preview   # Preview production build locally
```

## Stack

- **Vite** + **React** — SPA with client-side routing
- **Plus Jakarta Sans** + **IBM Plex Mono** — typography
- **Netlify** — hosting, redirects, headers

## Pages

- `/` — Landing page
- `/code` — Searchable project gallery with sort/filter
- `/coffee-chat` — Coffee chat scheduling
- `/contact` — Redirects to Google Form
- `/apps/*` — Standalone mini-apps (static HTML/JS)
